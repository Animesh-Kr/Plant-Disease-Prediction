# 🌿 AI Plant Health Diagnostic System
Identify 38 varieties of plant diseases with high-precision Deep Learning.

## 🚀 Features
- **Real-time Detection:** Upload a leaf image and get results in seconds.
- **Deep Learning:** Built using EfficientNetV2B0 for state-of-the-art accuracy.
- **Modern UI:** Streamlit-based interface with confidence metrics.

## 🛠️ Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/Animesh-Kr/Plant-Disease-Prediction.git](https://github.com/Animesh-Kr/Plant-Disease-Prediction.git)

2. Install dependencies:
   ```bash
   pip install -r requirements.txt

3. Run the application:
   ```bash
   streamlit run app/main.py

   
📂 Project Structure
app/: Contains the Streamlit web application.

model_training_notebook/: Contains the original training script and data processing logic.

trained_model/: Storage for the serialized Keras model.
